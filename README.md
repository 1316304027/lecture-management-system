# 講義管理システム

## 概要
講師・学生・管理者の3つの役割を持つ講義管理Webアプリケーションです。

## 技術スタック
- バックエンド：Java 21, Spring Boot
- データベース：PostgreSQL
- インフラ：Docker, AWS EC2
- バージョン管理：Git, GitHub

## 主な機能
- 管理者：ユーザー管理、授業管理、学生実績レポート一覧
- 講師：課題登録・編集・削除、教材アップロード
- 学生：出席一覧表示、教材ダウンロード、課題アップロード

## ローカル起動方法
1. リポジトリをクローン
2. データベースを起動
3. 以下のコマンドを実行
./mvnw spring-boot:run
